# sonar-Machine-learning
# Sonar Signal Classification

## Description
This project uses machine learning to classify sonar signals as either
a rock (R) or a mine (M).

## Dataset
The dataset contains 208 samples with 60 features representing sonar signal energy.

## Model Used
Logistic Regression from scikit-learn.

## Results
The model achieved approximately 80.95% accuracy on the test dataset.

## Tools
- Python
- Google Colab
- scikit-learn
